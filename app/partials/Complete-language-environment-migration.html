Earlier COBOL programs can run under several different runtime libraries, including Language Environment (LE). If any of your programs are still using old runtime library routines, you need to migrate them.

Select ALL actions that have been performed:

 

For CICS programs, when introducing Enterprise COBOL V5 or V6 programs into an application, take the following actions:
  • For statically CALLed CICS programs, relink with IGZEBST from LE with the PTFs for APAR PI33330 installed.
  • For dynamically CALLed CICS programs, install the PTFs for APAR PI25079 into SCEERUN.
